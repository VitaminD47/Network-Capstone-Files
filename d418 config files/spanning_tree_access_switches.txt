This is a re-usable config for RSTP on the exos access switches. It accounts for all vlans except the guest vlan.


disable stpd s0
configure stpd s0 mode dot1w
enable stpd s0
configure stpd s0 priority 
configure stpd s0 priority 32768


note : for radiology/clinic switches, use this line for the last line instead:
configure stpd s0 add vlan 10 ports 11
configure stpd s0 add vlan 20 ports 11
configure stpd s0 add vlan 30 ports 11


^ last line includes both 11/12 since they will be a LAG.
