##s0 is stpd name

To add tagged ports to stpd s0 (s0 is the domain), do this:

configure vlan Admin add ports 3 tagged stpd s0
configure vlan Radiology add ports 4,5 tagged stpd s0
configure vlan Clinical add ports 6,7 tagged stpd s0
configure vlan Guest add ports 8 tagged stpd s0

adjust port numbers as needed. These ports reflect core switch.
